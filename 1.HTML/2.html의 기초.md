## 1.3 Emphasis
내용을 좀더 특별하게 강조
```
<em>강조</em>
<strong>강조</strong>  
```

<br> : break. 줄바꿈. 남용 금지
+<br>과 <br/> : 각각 HTML5와 XHTML에서 선호. 혼용되는 것도 있음  

실제로 그렇게 보이는지는 중요하지 않음  
브라우저에게 중요한 내용이라는 것을 메세지를 전달했다는 것이 중요함

## 1.10 From_기본구조

### 문서 구조와 정보 위계가 명확히 보이는 의미구조에 맞는 코드  
div링크 남발하지 말자.  
iv는 브라우저가 정보의 위계질서 파악하는데 큰 도움을 주지 못함  
nav 태그를 소중히 씀. 한 페이지 당 보통 1개.  
+nav : 다른 웹페이지로 연결하는 링크 모음인 네비게이션 바를 만들 때 사용함  
```
  <nav> 
    <ul> 
      <li><a href="#">Home</a></li> 
      <li><a href="#">Blog</a></li> 
      <li><a href="#">Album</a></li> 
      <li><a href="#">Music</a></li> 
      <li><a href="#">Map</a></li> 
    </ul> 
  </nav>
```
